# PrintScore™ — AI Design Print Compatibility Analyzer

Made it with AI? Let's make sure it prints right.

**Live Demo:** https://innergclaw.github.io/printscore/

## Features

- **Instant Analysis** — Upload your design, get a print score in seconds
- **Client-Side Only** — All analysis happens in your browser, nothing is uploaded
- **PDF Report** — Download a shareable report with your score and recommendations
- **Free** — No signup required

## How It Works

1. **Upload** — Drop your PNG, JPG, or PDF file (max 10MB)
2. **Scan** — Instant analysis of resolution, color mode, layout, and format
3. **Download** — Get your print score + PDF report

## Print Score Breakdown

- **Resolution (40%)** — DPI and megapixel analysis
- **Size Suitability (30%)** — File size optimization
- **Color Mode (20%)** — RGB vs CMYK compatibility
- **Format (10%)** — File type assessment

## Score Tiers

- **90-100** — Print-Ready
- **75-89** — Great
- **60-74** — Needs Optimization
- **40-59** — High Risk
- **0-39** — Print Failure Likely

## Tech Stack

- Pure HTML, CSS, JavaScript (no build step)
- jsPDF for PDF generation
- Space Grotesk font (Google Fonts)

## Deployment

Deploy on GitHub Pages:

1. Push to GitHub repository
2. Go to Settings → Pages
3. Select source: `main` branch, `/ (root)`
4. Save
